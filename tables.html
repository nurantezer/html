Örneği daha iyi anlamak için tabloya kenarlıklar ekledik.

Kenarlık(border) eklemek için tablo, th ve td öğelerinde CSS border özelliğini kullanın:

table,th,td{ border: 1px solid black; }

Yukarıdaki örnekte olduğu gibi çift kenarlıklardan kaçınmak için, CSS border-collapse özelliğini daralt olarak ayarlayın.

table,th,td{ border: 1px solid black; border-collapse: collapse

Background color verme

th,td{ Background-color: #96D4D4 }

border-radius özelliği ile kenarlıklar köşeleri yuvarlatılır:Table eklersem tablonun içinde,eklemezsem yuvarlak hücreler verir.

th, td { border-radius: 10px }

border-style da değişik stiller var.

border-color ile tabloya sınır rengi verebilirsin.
